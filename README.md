# tmux
копировать .tmux.conf в ~/
в консоли
```
  git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
  tmux source ~/.tmux.conf
  
  # Next enter in tmux
  tmux
  
  #Inside tmux
  <prefix + I> - for install plugins
  
```
